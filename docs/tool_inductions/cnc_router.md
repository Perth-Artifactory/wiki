## Initial sessions

* Get the user comfortable with their CAM of choice (typically Autocad Fusion or Vectric Aspire)
* Step them through the [operation checklist](/tools/cnc/Swarf-XL/Operations-checklist)
  * For some users this is the first time they're meaningfully learning how to tighten things by hand (clamps). Overtightening is a real possibility here.
* Cover the basics of UCCNC
  * Loading files
  * Jogging
  * Explain how the the feed rate/spindle speed modifiers actually work (percentage based) and clarify their effect on plunge rates
  * + Everything mentioned in the operation guide
* Discuss the users intended project and tailor the training to it. (Ali brackets, wood carving etc)
* Clarify who can supervise them (Skip stages as appropriate):
  * You (If they're really early in the learning process)
  * Any trainer (If they still have some questions but have a pretty good handle on the machine)
  * Any operator (When they're comfortable running jobs but haven't been assessed yet)

## Final sign off

In order to be signed off on the router a user should meet the following competencies:

* Set up and export a job from their CAM software of choice
* Comfortable navigating the basic functions of UCCNC (Everything involved in running a basic job and adjusting the speed)
* Able to plan out and execute basic work holding
  * Types: (Strap, toe, vice, tape)
  * Positioning
  * Spoilboard
* A basic understanding of bit types and when they may be useful
  * Flute profile: (Upcut, downcut, compression, straight, 1-4F)
  * Tip profile (End mill, ball nose, v bit, drill)
* Able to pause/stop/e-stop a job
* Consistent PPE compliance
