Overview

This import lathe is currently being overhauled, with parts being re-machined to achieve smoother and more precise operation, and the main drive speed control being upgraded from a mechanical gearbox to fully electronic variable speed control.

Notes:

  • Function card H10001 is fitted.
    • 4 digital inputs.
    • 1 relay output.
    • 1 analog output.
    • Modbus.
  • The analog input can be swapped between 0-10 V or 0-10mA input mode using the jumper marked “AI1” behind the front cover.

 

 

Motor

Note this motor was sold as a “YX3-80M2-4”.

The motor as received has a different model no (GL-802-4 B3) and different technical specs.

  • In particular, the efficiency on the nameplate (73.0%) is much less than the eBay listing which quotes an efficiency of 82.3% for the YX3-80M2-4.
  • We probably received a different motor than what we ordered (or Conon Motor has pulled a fast one on us...)
